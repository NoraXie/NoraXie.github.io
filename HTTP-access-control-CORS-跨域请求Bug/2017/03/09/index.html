Cross Origin Resources Support

关于HTTP access control(CORS),官网有详细说明.这里不作缀述.只谈遇到的问题与解决方案
跨域请求时,如果响应的Content-Type不是下面三种之一,HTTP都认为是一个非简单请求

1 application/x-www-form-urlencoded
2 multipart/form-data
3 text/plain

在使用Spring的RESTfulAPI时,都是返回application/json,于是被当作一个非简单请求

非简单请求

非简单请求会在用户发送的GET/POST/HEAD请求之前,出于安全考虑,先自动向请求的目标地址先发送一个OPTIONS请求,只有当OPTIONS请求返回200时,才会去发送真实的请求.这个过程叫做preflight fetch.

一般通常的设置的方式有配置文件,Filter和拦截器.
按照Spring官网上的说法是,4.2+以上的版本都支持注解配置,参考这里
