生如蚁而美如神

发表于 2017-06-28 | 分类于 随笔 | | 本文总阅读量 次

这周开始已经不大记得时间,白天忘记几时日落.

一个下午的时间搭了个慢得unbelievable的ubuntu.开始搞os实验.各种命令终于可以看到大概的运行结果.

实在无法忍受ubuntu图形化界面巨大的图标,开始以为是ubuntu镜像的问题,于是去下了个1610的镜像,重新装,又是几个小时过去了.结果只解决了UI卡顿,图标依然丑陋无比.再次绕回原点,找到是pd的tools没有安装,还特么装不上,看完日志说是有一个文件找不到,显然是pd需要更新了.于是找资源,下载,安装,重启Ubuntu.一切搞定,24点了.

腰疼.
很容易知道是为什么.
我已经一周没出门.
一周都在椅子里看书,敲代码,看文档.
仍旧觉得脑子不够用.
现在只想把操作系统搞通.
感觉状态不好,断绝了与所有人的联系.
父母,朋友,同学.
想见不能见的人啊.
现在也不想见了.
长期的熬时间,让我面色暗黄,自己都讨厌自己.
做完操作系统的所有实验,我要出去见见世界.

网络安全
不会成真的梦
